# Projeto de Análise de Vendas com Python + Power BI

## Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de transformar uma base de dados de vendas em informações estratégicas por meio de:

Tratamento e preparação dos dados utilizando Python + Pandas
Criação de métricas analíticas
Desenvolvimento de um dashboard interativo no Power BI

O foco do projeto foi demonstrar todo o fluxo de análise de dados, desde a manipulação da base até a geração de insights visuais para apoio à tomada de decisão.

## Tecnologias Utilizadas
Python
Pandas
NumPy
Power BI
Git
GitHub

## Etapas do Tratamento de Dados

Durante o processamento da base, foram realizadas etapas como:

✔️ Limpeza e padronização dos dados
✔️ Tratamento de colunas e valores
✔️ Criação de métricas analíticas
✔️ Criação de novas colunas derivadas
✔️ Organização da base para visualização no Power BI
✔️ Mapeamento de regiões por loja
✔️ Cálculo de faturamento

# Dashboard Analítico

## Após o tratamento dos dados, foi desenvolvido um dashboard interativo no Power BI com análises como:

Faturamento total
Receita por loja
Receita por região
Produtos mais vendidos
Comparativo entre lojas
Evolução das vendas ao longo do tempo
Filtros Dinâmicos

# Perguntas que o Dashboard Responde

Qual loja possui maior faturamento?
Quais produtos mais geram receita?
Qual região apresenta melhor desempenho?
Como as vendas evoluem ao longo do tempo?
Qual o faturamento anual por loja?
Quais produtos possuem maior volume de vendas?
Como o desempenho varia entre os períodos?
